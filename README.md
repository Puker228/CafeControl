# CafeControl

## Python

В проекте используется python 3.11.

Для удобства лучше установить [uv](https://docs.astral.sh/uv/getting-started/installation/).

Далее необходимо синхронизировать зависимости:

```shell
uv sync
```

---

## Запуск

Для запуска достаточно запустить в терминале команду:

```shell
uv run main.py
```
